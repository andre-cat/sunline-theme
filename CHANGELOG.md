# Changelog

All notable changes to the Sunline theme will be documented in this file.

## [1.0.0] - 2025-01-27

### Added
- Initial release
- Complete light theme with blue/orange color palette
- Full UI theming (title bar, tabs, sidebar, panel, status bar)
- Token colors for popular languages (Python, JavaScript, TypeScript, HTML, XML, JSON, CSS, Java)
- Semantic token coloring support
- Symbol icons with semantic colors
- Merge editor colors (blue for current, orange for incoming)
- Notebook cells support
- Inlay hints styling
- Debug console and debug view colors
- Chart colors
- Git decorations (orange for modified files)
- Rainbow brackets (blue/orange palette)

### Color Palette

**Syntax:**
- Keywords: Black bold `#000000`
- Functions/Methods: Blue bold `#007AFF`
- Decorators/Regex: Denim blue `#5085B0`
- Self/Numbers/Constants: Light blue italic `#5A9FD4`
- Classes/Types/Booleans: Orange bold `#E67300`
- Parameters: Orange italic `#E67300`
- Strings: Olive `#635400`
- Docstrings: Light olive italic `#7A6B1A`
- Comments: Gray italic `#6c6c70`

**UI:**
- Primary: Blue `#007AFF`
- Accent: Orange `#E67300`
- Error: Red `#D93526`
- Success: Green `#32A852`
